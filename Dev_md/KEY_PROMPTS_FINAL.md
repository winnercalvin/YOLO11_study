## 🎯 Phase별 핵심 프롬프트

### 📘 Phase 1: 기본 시스템 구축

#### 프롬프트 1-1: 초기 개발 요청
```
파이썬 프로그램으로 yolo11을 사용해서 내가 입력해주는 그림 파일을 분석해서 
객체마다 사각형, 동그라미, 다각형을 표시해서 객체에 라벨링을 하는 프로그램을 개발해줘. 
https://github.com/aebonlee/251121 이 리포지토리에 커밋하고 푸시해줘.
```

**의도 분석**:
- ✅ YOLO11 사용 명시
- ✅ 다양한 도형으로 라벨링
- ✅ GitHub 연동 요구

**결과물**:
- `first/yolo_detector.py` (450 lines)
- 3가지 도형 자동 선택 알고리즘
- GitHub 초기 커밋

#### 프롬프트 1-2: 학습 자료 요청
```
.py 파일을 공부하며 테스트 하기 편하게 .ipynb 파일로도 만들어줘.
```

**의도**: 교육적 가치 추가
**결과**: 14개 섹션 Jupyter Notebook 튜토리얼

### 📗 Phase 2: 고급 기능 개발

#### 프롬프트 2-1: 정확도 향상
```

새로이 객체 80가지가 아닌 좀 더 정확하게 객체를 탐지 구분하는 프로그램을 구현해줘.
```

3. 세그멘테이션 추가

```

.ipynb 파일로 학습할 수 있게 주석도 달아서 소스 정리해줘.
```

**패턴 인식**: 반복되는 문서화 요청
